## 📞 SUPPORT & TROUBLESHOOTING

### npm install fails
**Solution**: Use `--legacy-peer-deps` flag or clear cache:
```bash
npm cache clean --force
npm install --legacy-peer-deps
```

### Port 3000 already in use
**Solution**: Kill the process or wait for it to close

### Electron window doesn't appear
**Solution**: Check React dev server started on port 3000

### Scanning is slow
**Solution**: Use Quick Scan for testing first
